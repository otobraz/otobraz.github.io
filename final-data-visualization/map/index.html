O Trabalho

Este trabalho prático compreendeu o desenvolvimento parcial do projeto da disciplina Visualização de Dados. Foi desenvolvido um heatmap para apresentar dados de conflitos armados que ocorreram entre nações entre 1500 e 2018.

Buscou-se responder as seguintes perguntas: Quantas conflitos e alianças foram formadas por cada país entre os anos de 1500 e 2018? e Para cada país, com quem eles tiveram conflitos e alianças? E quantas vezes?
