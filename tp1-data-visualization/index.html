O Trabalho

Este trabalho prático consiste na análise de dados das Eleições Gerais de 2014 do Brasil, geração de visualizações que apresentem as análises e explanação das visualizações

Foram geradas três visualizações. A primeira apresenta os partidos mais votados em cada estado brasileiro. A segunda mostra quantos votos cada cargo político recebeu. Por fim, a terceira apresenta a quantidade de votos recebidos pelos candidatos à presidência.

Visualizações Desenvolvidas

Partidos mais Votados

Esta visualização mostra através do mapa do Brasil, os partidos que receberam mais votos em cada estado. Foram gerados um mapa para o 1º turno e outro para o 2º turno.

Cargos mais Votados

Foi mostrado a quantidade de votos que cada cargo político recebeu. Foram utilizadas barras empilhadas para permitir a visualização dos votos do 1º e 2º turno de forma separada.

Votos para presidente

Foram utilizados gráficos de barra para visualizar a quantidade de votos dos candidatos a presidência. Outro gráfico foi gerado para visualizar a proporção de votos dos últimos candidatos.
